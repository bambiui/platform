## Usage

Add the design tokens to your global CSS:

```css
/* globals.css */
@import '@bambi-ui/theme/tokens.css';
@import '@bambi-ui/button/index.css';
```

Then import and use the component for your framework:

## CSS Tokens

All button styles are driven by CSS custom properties. Override any token in your own CSS to customize the button.

### Layout & Typography

| Token | Default | Value |
|---|---|---|
| `--bambi-button-gap` | `0.375rem` | — |
| `--bambi-button-border-width` | `1px` | — |
| `--bambi-button-line-height` | `1` | — |
| `--bambi-button-font-family` | `var(--bambi-font-sans)` | `system-ui, sans-serif` |
| `--bambi-button-font-weight` | `var(--bambi-font-weight-medium)` | `500` |
| `--bambi-button-font-size-sm` | `var(--bambi-text-xs)` | `0.75rem` |
| `--bambi-button-font-size-md` | `var(--bambi-text-sm)` | `0.875rem` |
| `--bambi-button-font-size-lg` | `var(--bambi-text-base)` | `1rem` |

### Padding

| Token | Default |
|---|---|
| `--bambi-button-padding-sm` | `0.375rem 0.75rem` |
| `--bambi-button-padding-md` | `0.5rem 1rem` |
| `--bambi-button-padding-lg` | `0.625rem 1.25rem` |
| `--bambi-button-padding-icon` | `0.5rem` |

### Radius

| Token | Default | Value |
|---|---|---|
| `--bambi-button-radius` | `var(--bambi-radius-md)` | `0.375rem` |
| `--bambi-button-radius-sm` | `var(--bambi-radius-sm)` | `0.25rem` |
