title: Theme Tokens
description: Global design tokens — colors, radius, typography, and shadows.
---

All tokens are CSS custom properties defined in `@bambiui/tokens/tokens.css`. Import once at the root of your app:

```css
@import '@bambiui/tokens/tokens.css';
```

Override any token in your own `:root` block to customise the theme globally:

```css
:root {
  --bambi-primary: oklch(60% 0.25 30);
  --bambi-radius-md: 0.5rem;
}

.dark {
  --bambi-primary: oklch(70% 0.25 30);
}
```

## Shadows

| Token | Light value |
|---|---|
| `--bambi-shadow-sm` | `0 1px 2px 0 oklch(0% 0 0 / 0.05)` |
| `--bambi-shadow-md` | `0 4px 6px -1px oklch(0% 0 0 / 0.1), 0 2px 4px -2px oklch(0% 0 0 / 0.1)` |
| `--bambi-shadow-lg` | `0 10px 15px -3px oklch(0% 0 0 / 0.1), 0 4px 6px -4px oklch(0% 0 0 / 0.1)` |

Shadow opacities are higher in dark mode (`0.3` / `0.4` / `0.4`) to compensate for darker backgrounds.

## Tailwind Integration

The token file ships an `@theme inline` block that maps Bambi tokens to Tailwind v4 CSS variables. No extra config is needed — importing `tokens.css` is enough:

```css
/* globals.css */
@import '@bambiui/tokens/tokens.css';
```

```html
<!-- Tailwind classes resolve to Bambi tokens automatically -->
<div class="bg-primary text-primary-foreground rounded-md shadow-md">
  Hello
</div>
```
