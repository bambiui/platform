## Installation

Add the button source and shared styles to your app:

```sh
npx @bambiui/cli init
```

<Tabs syncKey="framework">
  <TabItem label="React">
    ```sh
    npx @bambiui/cli add button --framework react
    ```
  </TabItem>
  <TabItem label="Svelte">
    ```sh
    npx @bambiui/cli add button --framework svelte
    ```
  </TabItem>
  <TabItem label="Vue">
    ```sh
    npx @bambiui/cli add button --framework vue
    ```
  </TabItem>
  <TabItem label="Astro">
    ```sh
    npx @bambiui/cli add button --framework astro
    ```
  </TabItem>
</Tabs>

Import the generated token file once from your app's global stylesheet. Component CSS is installed next to the component source and imported by the component.

```css
@import './styles/bambi.css';
```

## ApiReference

All framework packages share the same prop API.

| Prop | Type | Default | Description |
|---|---|---|---|
| `intent` | `'primary' \| 'secondary' \| 'danger' \| 'success' \| 'warning'` | `'primary'` | Semantic meaning |
| `appearance` | `'solid' \| 'outline' \| 'ghost' \| 'link'` | `'solid'` | Visual treatment |
| `size` | `'sm' \| 'md' \| 'lg' \| 'icon'` | `'md'` | Button size |
| `loading` | `boolean` | `false` | Shows spinner and prevents interaction |
| `disabled` | `boolean` | `false` | Disables the button |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | HTML button type |

All other HTML button attributes (e.g. `onClick`, `aria-*`, `style`, `class`) are forwarded to the underlying `<button>` element.
