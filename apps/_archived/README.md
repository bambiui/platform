# Archived Apps

Everything under `apps/_archived` is inactive reference material only. These apps are not current workspace, build, registry, or CLI targets.

Do not use old `@bambiui/components`, `@bambiui/tokens`, docs, studio, archived marketing, or legacy runtime-package references as guidance for the active architecture. Current source of truth is the DOM Protocol, CLI-first flow: `packages/core` internal contracts/controllers, `packages/generator` build-time generated artifact creation, `packages/registry` self-contained React output and CSS, `packages/cli` artifact copying, and `apps/www` registry asset hosting.
