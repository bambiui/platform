---
title: Button Tokens
description: CSS custom properties that control every aspect of the Button component.
---

Button tokens are defined alongside the global theme in `@bambi-ui/theme/tokens.css` and consumed by `@bambi-ui/button/index.css`. All tokens are namespaced `--bambi-button-*`.

Override them in `:root` (global) or on a selector (scoped):

```css
/* global override */
:root {
  --bambi-button-radius: 9999px;
  --bambi-button-hover-opacity: 0.85;
}

/* scoped to a container */
.my-card {
  --bambi-button-radius: 0;
  --bambi-button-primary-bg: oklch(55% 0.2 160);
}
```

## Border Radius

| Token | Default | Resolved value |

| `--bambi-button-radius` | `var(--bambi-radius-md)` | `0.375rem` |

## Examples

### Pill buttons globally

```css
:root {
  --bambi-button-radius: 9999px;
}
```

### Brand-coloured buttons

```css
:root {
  --bambi-button-primary-bg: oklch(55% 0.2 160);      /* green */
  --bambi-button-primary-foreground: oklch(98% 0 0);
}
```

### Slower hover transition

```css
:root {
  --bambi-button-transition: opacity 0.3s ease, background-color 0.3s ease;
}
```

### Compact padding

```css
:root {
  --bambi-button-padding-md: 0.375rem 0.75rem;
  --bambi-button-padding-lg: 0.5rem 1rem;
}
```
