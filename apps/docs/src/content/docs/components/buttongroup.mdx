---
title: ButtonGroup
description: Group related buttons as attached segments or spaced controls in React, Svelte, Vue, or Astro.
---

import { Tabs, TabItem, Code } from "@astrojs/starlight/components";
import Button from "@bambiui/components/button/astro";
import ButtonGroup from "@bambiui/components/buttongroup/astro";

## Installation

```sh
npx bambiui init
npx bambiui add button
npx bambiui add buttongroup
```

ButtonGroup copies a small wrapper component and local CSS into your project. It does not depend on a runtime bambiui package.

Then import the components:

## API Reference

| Prop          | Type                         | Default        | Description                            |
| ------------- | ---------------------------- | -------------- | -------------------------------------- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction of the grouped controls      |
| `attached`    | `boolean`                    | `true`         | Removes gaps and merges button borders |
| `role`        | `string`                     | `'group'`      | ARIA grouping role                     |

All other HTML div attributes (`aria-*`, `style`, `class`) are forwarded to the wrapper. React uses `className`; Svelte, Vue, and Astro use `class`.
